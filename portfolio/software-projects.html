Autonomous Vehicle

An autonomous vehicle operating on ROS. Includes self driving with lane detection, crosswalk detection, sign detection, with a self built AI model.

The autonomous vehicle includes the following sensors: a LiDAR, a camera, as well as an ackermann steering mechanism. The car itself uses a LiDAR camera fusion for object detection as well as street detection, and uses a simple pID controller for actual steering.

Remote Copy Program

Network file transfer utility written in C with comprehensive error handling, progress tracking, and support for large file transfers across different platforms.

Implemented a protocol that has error checking and uses a sliding window with an interchangeable window size that mimics TCP very closely. Simulated packet dropping and bit switching when testing and uses a gethostbyname library as well as sockets API in order to connect client to server.

Inverted Pendulum Robot

Self-balancing robot built from scratch with custom control systems. Implemented mathematical modeling and PID control algorithms for stable operation.

A classic robotics challenge that showcases knowledge of control systems. Implemented mathematical modeling and PID control algorithms for stable operation. One of my favorite projects, instead of diving in and doing engineer things, a mathematical model was created to optimize the best PID values, how to program the microcontroller, and simulate the entire system. This cut down on a lot of time that would've been spent tweaking PID values and provided invaluable insight on how to mathematically model real world systems.
